# OpenCode Platform

Plataforma personal de trabajo construida exclusivamente sobre OpenCode.

## Principios

- Simplicidad antes que complejidad.
- Solución oficial antes que creación propia.
- Calidad antes que cantidad.
- Validación antes que incorporación.
- Evolución en lugar de duplicación.

## Estructura

- `.opencode/`: configuración oficial de OpenCode para el proyecto.
- `docs/`: arquitectura y decisiones relevantes.
- `experimental/`: elementos todavía no validados.
- `scripts/`: automatización reproducible de la plataforma.
